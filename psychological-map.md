## How this file works

- This file is the **primary and authoritative personal model**. `patterns/`, `sessions/`, `journal/`, and `experiments/` exist to feed it. They never replace it.
- Updates happen only via the `update-map` command (`.ai/commands/update-map.md`), or by deliberate edit reviewed against the rules below.
- **Promotion rule:** an entry is added only when there is **at least 3 distinct observations on different days** supporting it. Single events live in `journal/`. Repeated observations live in `patterns/`. Well-supported patterns live here.
- **Separate observation from interpretation.** Every claim is structured as: what is observed, what is the current hypothesis, and how confident it is.
- **Cite evidence.** Every non-trivial entry links to specific files in `patterns/`, `sessions/`, or `experiments/`.
- **Supersede, don't overwrite.** When an entry becomes outdated, move the old version to `archive/psychological-map-<date>.md` and write the replacement with a back-reference.
- **No reassurance.** This file is for accurate self-modeling, not for making me feel better.

## Entry format

```markdown
### <Short name>

- **Observation:** <what is repeatedly observed, plain language>
- **Interpretation:** <current best hypothesis — marked as hypothesis>
- **Confidence:** low / medium / high
- **Sources:** `patterns/...`, `sessions/...`, `experiments/...`
- **Last reviewed:** YYYY-MM-DD
```
