# Open Questions

## Purpose

A living list of questions about myself that I do not yet have answers to, but want to keep tracking. Open questions are the seedbed for behavioral experiments. A good open question is one whose answer would change behavior.

## Update rules

- Add questions when sessions or pattern analyses surface uncertainty that recurs.
- Each question gets a **status**: `open`, `investigating`, `partial answer`, `closed`.
- Questions are not deleted when answered — they are moved to a `## Closed` section at the bottom, with a one-line note on what was learned and where the answer lives (e.g., a session, a closed experiment, a `core/` entry).
- Review every 4–6 weeks: prune duplicates, escalate questions ripe for experiment, mark stale ones.
- Phrase questions as **questions**, not problems. "Is X true?" not "X is bad."

## Entry template

```markdown
### Q: <the question>

- **Status:** open / investigating / partial answer / closed
- **Why it matters:** <what changes if I learn the answer>
- **What would answer it:** <observation, conversation, experiment>
- **Related:** <files in `sessions/`, `patterns/`, `experiments/`>
- **Opened:** YYYY-MM-DD
- **Last touched:** YYYY-MM-DD
```

## Example

```markdown
### Q: Does my pre-1:1 anxiety drop when I prepare the agenda the night before?

- **Status:** investigating
- **Why it matters:** If yes, "anticipation anxiety" might be largely about preparation control, not evaluation. If no, the fear is about being seen, not being unready.
- **What would answer it:** A 4-week experiment alternating prepared vs. unprepared 1:1s, rating anxiety night-before on a 1–5 scale.
- **Related:** `experiments/2026-06-01-agenda-night-before.md`, `patterns/anxiety.md#anticipation`
- **Opened:** 2026-05-30
- **Last touched:** 2026-06-09
```
