# Patterns: Anxiety

## Purpose

Detailed catalog of recurring anxiety patterns. This file sits between raw observation (in `sessions/`, `journal/`) and the high-level model (`core/psychological-map.md`). A pattern earns a place here once it has recurred across multiple sessions or days. It earns a place in `core/` only when it is well-supported and stable.

## Update rules

- A pattern is added only after **at least 3 distinct observations on different days**.
- Every pattern entry has: trigger, body signal, behavior, time course, evidence, counter-evidence.
- **Counter-evidence is required.** A pattern without a search for counter-examples is suspect.
- Avoidance is itself a pattern. If something is being systematically dodged in sessions, name it here.
- This file is **not** an inventory of every anxious moment. Single events belong in `sessions/` or `journal/`. Patterns are the recurring shape.
- When a pattern weakens, mark `confidence: lowered` rather than deleting. When it disappears, move to `archive/`.

## Entry template

```markdown
### <Pattern name>

- **Trigger:** <what situation reliably precedes it>
- **Body signal:** <physical sensations and where they appear first>
- **Thought content:** <the typical mental content, if any — note that body often precedes thought>
- **Behavior:** <what gets done or avoided as a result>
- **Time course:** <onset, peak, decay — minutes, hours, days>
- **Evidence:** `sessions/...`, `journal/...`
- **Counter-evidence:** <situations where this would have been expected but did not occur>
- **Confidence:** low / medium / high
- **Linked experiments:** `experiments/...`
- **Last reviewed:** YYYY-MM-DD
```

## Example

```markdown
### Anticipation spike before social evaluation

- **Trigger:** Scheduled 1:1s, performance reviews, social events with new people. Reliable 12–24h before.
- **Body signal:** Jaw tension first, then shallow chest breathing. Often noticed in the evening.
- **Thought content:** Rehearsing what could go wrong. Replays of past similar events.
- **Behavior:** Avoidance of agenda prep. Late-night doomscrolling. Sometimes a cancellation excuse drafted but not sent.
- **Time course:** Peaks the night before, drops sharply once the event begins.
- **Evidence:** `sessions/2026-05-18-pre-1on1-spike.md`, `sessions/2026-05-30-review-eve.md`
- **Counter-evidence:** Peer 1:1s with M and J have not produced the spike — suggests the "evaluation" axis matters more than "social interaction".
- **Confidence:** medium
- **Linked experiments:** `experiments/2026-06-01-agenda-night-before.md`
- **Last reviewed:** 2026-06-09
```
