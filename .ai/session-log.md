# Session Log

Chronological index of every session in `sessions/`. Newest entries at the top.

## Purpose

A flat, scrollable index so an AI (or future-me) can quickly orient:

- "What have we been working on lately?"
- "When was the last time we touched anxiety?"
- "Which sessions fed which experiment?"

## Update rules

- Add an entry **every time** a new session file is created in `sessions/`.
- One line per session. If you need more, the session file itself is the place.
- Tag the dominant theme(s) in brackets: `[anxiety]`, `[adhd]`, `[relationships]`, `[self-esteem]`, `[work]`, `[meta]`, `[experiment]`.
- If the session produced a map update or experiment, mark it: `→ map`, `→ experiment:<name>`.

## Format

```
- YYYY-MM-DD — <topic> — <one-line gist> [tags] [→ map] [→ experiment:<name>]
```

## Example

```
- 2026-06-12 — pre-1on1-spike — anxiety always peaks the night before 1:1s; noticed avoidance of agenda prep [anxiety][work] → experiment:agenda-night-before
- 2026-06-09 — saturday-flat — flat mood on unstructured weekends; hypothesis: lack of external scaffolding [adhd][self-esteem]
- 2026-06-05 — partner-feedback-loop — replayed conversation with M; defensive reaction tracked back to evaluation fear [relationships][self-esteem] → map
```
